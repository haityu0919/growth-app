# 管理者通知の確認方法

開発中は、管理アプリに送る通知の元データを **JSONファイル** に保存します。あとで管理アプリを作るときは、このデータを画面に表示します。

## 保存場所

```text
C:\Users\matsuda\Documents\growth-app\token-server\data
```

| ファイル | 内容 |
|----------|------|
| `inquiries.json` | 問い合わせ |
| `admin-call-requests.json` | 管理者と通話したいリクエスト |

## 確認手順

1. `token-server\サーバー起動.bat` を起動したままにする
2. `mobile\QRで起動.bat` でアプリを起動する
3. ホーム画面で **問い合わせ** または **管理者と通話** を押す
4. 送信する
5. `token-server\data` フォルダを開く
6. `inquiries.json` または `admin-call-requests.json` をメモ帳で開く

## 管理アプリが見るAPI

管理アプリを作るときは、次のAPIを定期的に確認すれば通知一覧を表示できます。

| API | 内容 |
|-----|------|
| `GET /api/inquiries` | 問い合わせ一覧 |
| `GET /api/admin-call-requests` | 管理者通話リクエスト一覧 |

例:

```text
http://192.168.0.245:3001/api/admin-call-requests
```

```json
[

    "id": "自動生成ID",

  }
]
```
