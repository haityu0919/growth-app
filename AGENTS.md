# だれトーク（growth-app）

不特定の相手と**音声だけ**で話せるモバイルアプリ（Expo + LiveKit）。

## 再開

1. `HANDOFF_だれトーク.md` を読む
2. 直近の `TEAM_REPORT_*.md` を読む
3. 判断が必要なら **5名チーム** で議論してから結論（`.cursor/rules/daretalk-team.mdc`）

## チーム（5名）

| 名前 | 役割 |
|------|------|
| ジョブズ | 統括 |
| はるか | リサーチ |
| 風間 | 冒頭構成 |
| 匠 | UI/UX |
| こと | 文案 |

## コード

- アプリ: `mobile/`
- トークンサーバー: `token-server/`
